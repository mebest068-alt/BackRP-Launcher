# BackRP Launcher

Electron-лаунчер для SA:MP-проекта BackRP.

## Возможности

- Тёмный интерфейс на русском языке в стиле GTA San Andreas RolePlay.
- Отображение IP сервера `188.127.241.8:1464` и текущего онлайна через UDP-запрос SA:MP.
- Выбор и сохранение игрового ника.
- Запуск `samp.exe` с автоматическим подключением к серверу.
- Проверка наличия `gta_sa.exe` и `samp.exe`.
- Кастомная шапка окна, экран загрузки, уведомления и анимации.
- Автообновление через GitHub Releases на базе `electron-updater`.

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

В режиме разработки автообновление не устанавливается, но интерфейс покажет соответствующее сообщение.

## Настройка GitHub Releases

В `package.json` замените поля:

```json
"owner": "YOUR_GITHUB_USERNAME",
"repo": "BackRPLauncher"
```

на владельца и название вашего GitHub-репозитория. Для публикации обновлений создавайте новый релиз с версией выше текущей версии из `package.json`.

## Сборка `.exe`

```bash
npm run dist
```

Готовый установщик появится в папке `dist`. Для корректной иконки положите файл `assets/icon.ico` или измените путь `build.win.icon` в `package.json`.

## Как работает запуск игры

Лаунчер ищет `gta_sa.exe` и `samp.exe` в выбранной пользователем папке и в нескольких популярных директориях. Перед запуском выбранный ник записывается в `HKCU\Software\SAMP\PlayerName`. При нажатии `Играть` запускается:

```bash
samp.exe 188.127.241.8:1464
```

Если игра установлена в другой папке, нажмите `Папка GTA` и выберите директорию, где лежат оба файла.

## Файлы проекта

- `package.json` — зависимости, команды запуска и конфигурация `electron-builder`.
- `main.js` — главное окно, автообновления, SA:MP query, проверка игры и запуск.
- `preload.js` — безопасный мост между Electron main process и интерфейсом.
- `renderer.js` — логика интерфейса, уведомления, прогресс обновлений.
- `index.html` — разметка лаунчера.
- `style.css` — визуальный стиль, адаптивность и анимации.
